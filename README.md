# Car and Bike Rental Website
A full-stack Car and Bike Rental website where the customer can browse and rent cars and bikes offered by the rental agency. The vehicle rent data will be stored in the database so that the agency can deliver the vehicle and booking of the same vehicle on the same date is also prohibited.
